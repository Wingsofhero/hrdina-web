# Hrdina Web

Prezentačný React + Vite web pre službu **Hrdina Web**.

## Spustenie

```bash
npm install
npm run dev
```

Rýchly lokálny náhľad bez inštalácie závislostí otvoríš cez `local-preview.html`.

## Build pre Vercel

```bash
npm run build
```

Projekt používa React, Vite a čisté CSS bez zbytočných UI knižníc.
